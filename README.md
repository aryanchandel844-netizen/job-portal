# 🏢 Job Portal

> A real-world job portal where employers post jobs and candidates apply — built from scratch using Django REST + React.js

I built this project to strengthen my full-stack skills. It handles real use cases like JWT-based login, role-based access (seeker vs employer), and live job filtering.

---

## 🚀 What This Project Does

- Job seekers can **browse, filter, and apply** for jobs
- Employers can **post jobs and manage applications**
- Admin can **manage everything** from Django admin panel
- All APIs are **JWT secured**

---

## 🛠️ Tech I Used

| Layer | Technology | Why I Chose It |
|-------|-----------|----------------|
| Backend | Django + DRF | Fast API development |
| Frontend | React.js + Vite | Component-based UI |
| Styling | Tailwind CSS | Utility-first, fast |
| Database | MS SQL Server | Already knew it from college |
| Auth | JWT (SimpleJWT) | Industry standard |
| Testing | Postman | API validation |

---

## 💡 Features I Built

### For Job Seekers
- Register & Login with JWT tokens
- Browse all active jobs
- Filter by job type, experience level & location
- Apply with a cover letter
- Track application status (Pending → Accepted/Rejected)

### For Employers
- Post new job listings
- View all received applications
- Accept or reject candidates

### For Admin
- Full control via Django Admin Panel
- Manage users, jobs, applications

---

## 🧠 What I Learned Building This

- How JWT authentication actually works end-to-end
- How to connect Django REST APIs with React frontend
- CORS configuration between two different servers
- Role-based access control (employer vs seeker)
- MS SQL Server integration with Django using ODBC
